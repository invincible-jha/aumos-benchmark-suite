# Security Policy — aumos-benchmark-suite

## Reporting a Vulnerability

Report security vulnerabilities to security@muveraai.com.

Do NOT open a public GitHub issue for security vulnerabilities.

## Scope

This service exposes benchmark data which may include proprietary competitor
baseline scores. Access to baselines and reports is protected by JWT
authentication and tenant-level RLS isolation.

## Supported Versions

Only the latest release receives security patches.
